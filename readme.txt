== Installation ==

1. Upload Zip file lr-mailchimp-subscription to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Place `<?php echo do_shortcode('[LRM_WP_MAILCHIMP] ');?>` in your templates files or You can add [LRM_WP_MAILCHIMP] to post,pages.
